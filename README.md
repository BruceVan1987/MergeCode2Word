MergeCode2Word
==============

将源代码拷贝到word文档

这个程序是为了方便申请软件著作权时候将所有代码拷贝到一起，申请时需要提交的源程序代码要求60页，每页至少50行，虽然手工操作也不算难事，但是为了方(tou)便(lan)，就想通过软件来完成。

这是一个swing程序，使用了[POI库](http://poi.apache.org/)，目前的功能很简单，支持将c,java,c#等源代码拷贝到word文档。

缺点：

1. 无法判断要生成的word文档的页数
2. 代码缩进效果在word文档中无效
